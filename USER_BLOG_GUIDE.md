# User Blog Management Guide

## Fitur Blog untuk User

### Semua User yang Login Bisa:
- ✅ Membuat blog post dengan featured image
- ✅ Edit post sendiri
- ✅ Delete post sendiri
- ✅ Melihat semua post sendiri di "My Posts"
- ✅ Upload featured image (max 5MB)
- ✅ Publish atau save as draft

### Admin Bisa:
- ✅ Semua fitur user
- ✅ Edit/delete post user lain
- ✅ Manage user roles
- ✅ Moderate messages

## Cara Membuat Blog Post

### 1. Create New Post
1. Login ke akun Anda
2. Klik "My Posts" di navigation bar
3. Klik "Create New Post"
4. Upload featured image (optional):
   - Click to upload atau drag & drop
   - Max 5MB (JPG, PNG, GIF)
   - Preview muncul otomatis
5. Isi Title dan Content
6. Centang "Publish immediately" untuk publish langsung
7. Klik "Create Post"

### 2. Manage Your Posts
1. Klik "My Posts" di navigation
2. Lihat semua post Anda (published & draft)
3. Setiap post menampilkan:
   - Featured image (jika ada)
   - Title dan excerpt
   - Status (Published/Draft)
   - Tombol Edit dan Delete

### 3. Edit Post
1. Di "My Posts", klik tombol "Edit"
2. Update featured image, title, atau content
3. Toggle publish status
4. Klik "Update Post"

### 4. Delete Post
1. Di "My Posts", klik tombol "Delete"
2. Confirm deletion
3. Post dan featured image akan terhapus permanent

## Post Visibility

### Published Posts
- Terlihat di homepage (Blog)
- Terlihat di profile Anda
- Terlihat oleh semua orang

### Draft Posts
- Hanya terlihat di "My Posts"
- Tidak muncul di homepage
- Hanya Anda yang bisa lihat

## Navigation Menu

### User Biasa
- **Blog**: Homepage dengan semua published posts
- **Dashboard**: Dashboard user
- **My Posts**: Manage post sendiri
- **Messages**: Lihat approved messages
- **Profile**: View/edit profile

### Admin
- **Blog**: Homepage
- **Dashboard**: Dashboard
- **My Posts**: Manage semua post
- **Admin Messages**: Moderate messages
- **Users**: Manage user roles
- **Profile**: View/edit profile

## Tips
1. **Featured Image**: Gunakan gambar landscape (1200x630px) untuk hasil terbaik
2. **Title**: Buat title yang menarik dan SEO-friendly
3. **Draft**: Save as draft untuk review sebelum publish
4. **Edit**: Anda bisa edit post kapan saja
5. **Delete**: Hati-hati, delete permanent dan tidak bisa di-undo

## Permissions
- User hanya bisa edit/delete post sendiri
- Admin bisa edit/delete semua post
- Tombol edit/delete hanya muncul jika Anda punya permission
